About this project:-

In this project, we aim to practice using R markdown presentation to create document and practice to make data visualization using library plotly.

Data used in this study:-
- We used data of spatial rainfall of all province in Thailand in year 2025 to practice and visualize data, data store in .csv format. Data is downloaded from website of Open
Goverment data of Thailand (<https://data.go.th/dataset/spatial-rain>). 
- We used GeoJSON data of Thailand to help visualisation in choropleth map, data store in .geojson format.

Files:-
- [Rainfall-amount-of-each-province-in-Thailad-December2025.Rmd](Rainfall-amount-of-each-province-in-Thailad-December2025.Rmd) : R markdown presentation file describe detail of the project, should be run in R.
- [Rainfall-amount-of-each-province-in-Thailad-December2025.html](Rainfall-amount-of-each-province-in-Thailad-December2025.html): HTML file generated from R markdown presentation which describe detail for this project.
- [simplified_provinces_geojson.geojson](simplified_provinces_geojson.geojson): GeoJSON data of all province in Thailand in .geojson format. 
- [spatial-rain-hii.csv](spatial-rain-hii.csv): spatial rainfall data of all province in Thailand from 2018-2026.
